# Student Grade Prediction - Визуализация данных

## 📊 Описание проекта

Проектная лабораторная работа по дисциплине **"Визуализация данных"** на основе датасета **Student Grade Prediction** из Kaggle.

Проект охватывает полный цикл аналитики:
- 🔍 **EDA** (разведочный анализ данных)
- 🧹 **Очистка данных** и предварительная обработка
- 📈 **Статистический анализ** признаков
- 📊 **Визуализация** (графики, диаграммы, корреляционные матрицы)
- 🤖 **ML-модель** для прогнозирования итоговой оценки
- 📱 **Интерактивный дашборд** на Streamlit

## 🎯 Исследовательские вопросы

1. Какие факторы наиболее сильно влияют на итоговую оценку студента (G3)?
2. Существует ли связь между половой принадлежностью и успеваемостью?
3. Как время обучения влияет на итоговые оценки?
4. Какие студенты попадают в группу риска (низкие оценки)?
5. Можно ли построить модель для прогнозирования итоговой оценки?

## 🚀 Инструкция по запуску

### 1. Клонирование репозитория

```bash
git clone https://github.com/AlikhanS669/student-grade-visualization.git
cd student-grade-visualization
```

### 2. Создание виртуального окружения

```bash
python -m venv venv

# Windows
venv\Scripts\activate

# macOS/Linux
source venv/bin/activate
```

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 4. Загрузка датасета

Скачайте датасет с Kaggle:
- Ссылка: https://www.kaggle.com/datasets/dipam7/student-grade-prediction
- Поместите файл `student-grades.csv` в папку `data/raw/`

### 5. Запуск Jupyter Notebook для EDA

```bash
jupyter notebook notebooks/01_eda.ipynb
```

### 6. Запуск интерактивного дашборда

```bash
streamlit run app/streamlit_app.py
```

Откройте браузер на `http://localhost:8501`

## 📈 Ключевые выводы

### EDA Findings:
- Среднее значение G3: 11.91 (0-20)
- Нет пропущенных значений
- Нет дубликатов
- Левостороннее распределение оценок

### Statistical Analysis:
- Девочки в среднем выше на 1.2 балла (p < 0.05)
- Студенты, учащиеся >20ч/неделю, получают 15.59 баллов
- Студенты, учащиеся <2ч/неделю, получают 10.21 балла
- Top-5 коррелирующих признаков: G2 (0.923), G1 (0.841), failures (-0.391), age (-0.248), studytime (0.097)

### ML Model Results:
- **Random Forest R² = 0.821** (лучше, чем Linear Regression)
- Top-3 признака: G2 (25.3%), G1 (18.7%), failures (12.4%)
- RMSE = 1.345, MAE = 1.089
- Модель хорошо работает и может помочь в раннем выявлении студентов в группе риска
